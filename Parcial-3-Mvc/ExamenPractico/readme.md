Examen Práctico — Parcial 3
Objetivo
Implementar un CRUD de torneos de basketball en PHP usando el patrón de diseño MVC, PDO, Programación Orientada a Objetos y encriptación de contraseñas, conectado a una base de datos MySQL.

Tecnologías utilizadas
PHP 8+
MySQL
PDO
Bootstrap 5
Font Awesome 6
VSCode
Git + GitHub
Instrucciones de ejecución
Tener XAMPP instalado y corriendo (Apache + MySQL)
Clonar el repositorio en la carpeta htdocs de XAMPP
Crear la base de datos proyecto en phpMyAdmin con una tabla torneos
Abrir el navegador en http://localhost/examen-practico/views/admin/admin.php
Funcionalidades
Registrar torneos con nombre, organizador, patrocinadores, sede, categoría y premios
Listar todos los torneos registrados
Ver el detalle de un torneo
Editar la información de un torneo
Eliminar torneos con confirmación mediante modal
Encriptación de contraseña del organizador con password_hash
