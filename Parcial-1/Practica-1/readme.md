Práctica 1 - Clases y Encapsulamiento en PHP
Objetivo
Aplicar los fundamentos de la Programacion Orientada a Objetos en PHP creando una clase con atributos privados, constructor y metodos de acceso.

Descripción
Se hizo la clase Usuario con dos atributos privados: nombre y correo.
Incluye un constructor, metodos getter (getNombre, getCorreo) y setter (setNombre, setCorreo).

Ejecución
Iniciar Apache en XAMPP.
